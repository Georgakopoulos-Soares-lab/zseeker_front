ZSeeker Web Interface

Introduction

The ZSeeker web interface is designed to provide a simple and accessible way to use the ZSeeker command-line tool. This web application is built using the Gin framework for backend operations in Go, while HTML, CSS, and JavaScript are used for the frontend. The main purpose of this interface is to make ZSeeker’s functionality easy to access. Users can submit jobs, inspect results, download data, and filter Z-DNA outputs directly from the web interface. Additionally, the application includes visual tools to help users understand Z-DNA regions better. For more information on the tool’s algorithms, visit the About/Contact page, and for guidance on using specific features, see the Help page.

Job Submission

This section displays the "Job Submission" page, where users can configure their analysis settings. Each parameter can be modified by clicking the input fields or typing directly. After setting the desired parameters, users can upload a FASTA file or paste sequences in the provided text area. Once ready, clicking "Submit" will send the job to the server for processing.



Results

After submitting a job, users will see the Z-DNA results displayed in a table format. The processing time may vary depending on file size and parameter settings. Results can be downloaded as a CSV file by clicking the "Download" button, and users can filter or sort the data for easier viewing. Additionally, users can select specific sequences from the input FASTA file by using the "Select Sequences" dropdown. This dropdown also shows the average Z-score and GC content for all detected sequences above the threshold. When users hover over a sequence, visual indicators highlight dinucleotides that contribute positively to the score in green, while interruptions that reduce the score appear in red.



Visualizations

Each time a new job is submitted, two visualizations are generated. On the left, a scatter plot shows how sequence length affects Z-DNA scores, with a color scale for identifying outliers quickly. The second visualization is a box plot that gives a summary of Z-score values, including minimum, median, maximum, and first and third quartiles. These visualizations make it easier to interpret and compare the Z-DNA scores across different sequences.
